<Update label="Anaconda 2.0.1 (Jun 12, 2014)" description="">
    Fixes:
    
    * added missing libpython (the so-called MinGW import library) for
      Python 3.4
    * in order to make pandas.io.excel work, we downgraded openpyxl from 2.0.2
      to 1.8.5
    * added missing idle script on Windows
    
    Updates:
    
    * conda from 3.5.2 to 3.5.5
    * conda-build from 1.3.3 to 1.3.5
    * numba from 0.13.1 to 0.13.2
    * pandas from 0.13.1 to 0.14.0
    * python from 2.7.6 to 2.7.7
    * scikit-image from 0.9.3 to 0.10.0
