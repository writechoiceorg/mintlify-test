<Update label="Anaconda 1.4.0 (Mar 12, 2013)" description="">
    Many more packages are now supported on Python 3.3. In addition, we
    have added Python 3 support on Windows, such that now Python 2.6, 2.7
    and 3.3 is supported across all platforms.
    In addition, we have redesigned the Windows installer, which was suffering
    from a number of problems, in particular the lack of uninstall functionality
    and the GUI not working on some systems.
    
    updated:
    
    * conda from 1.3.5 to 1.4.2
    * numba from 0.6.0 to 0.7.0
    * numpy from 1.7rc1 to 1.7.0
    * scikit-image from 0.7.1 to 0.8.2
    
    fixed:
    
    * pytables on Windows
    
    other notes:
    
    * Linux: a ATLAS package is now available (conda install atlas).
      In addition the site.cfg in the numpy.distutils has been updated,
      such that other packages (e.g. scipy) which use the atlas build
      configuration can be build against the (non-MKL linked) numpy in
      Anaconda.
